# Embedded Systems Programming with C++

This is part of the code [repository](https://github.com/PacktPublishing/Embedded-Systems-Programming-with-Cxx) for<br>
**Embedded Systems Programming with C++** by [Packt Publishing](https://www.packtpub.com/)<br>
**Learning from scratch to configure, develop, test and monitor.**

This folder contains examples from **Chapter 4: Configuring the embedded device local access and deploy**.

## Examples

* **Example01:** Creating an Yocto distribution with SSH
* **Example02:** Creating an Yocto distribution with RSYNC
* **Example03:** Acessing devices via SSH
* **Example04:** Sending commands to devices via SSH
* **Example05:** Transferring files to devices via SCP
* **Example06:** Transferring files to devices via RSYNC
